Problema 03

En este ejercicio visualizaremos el Teorema de Moivre-Laplace (TML). Para p=0.1 y A = {5, 10, 20, 50, 100, 500}, grafica lo siguiente:

  1. Sobre la misma figura, grafica la funcin de masa g(x) de una distribucin Binomial(n,p) y la funcin de densidad f(x) de una distribucin Normal(np,npq), para todo ``n en A. (i.e.presenta las seis figuras).

  2. Haz lo mismo que en el inciso anterior, pero ahora para las funciones de distribucin acumuladas de las binomiales y normales anteriores.

  3. Cul es la relacin entre as figuras anteruires y el TML? Cambia el resultado si uno toma p=0.5 o p=0.9?

Para el inciso a): El siguiente cdigo genera seis figurar para comparar la Distribucin Binomial con su aproximacin a la Normal para cada valor de nuestras n.

# Parmetros
p <- 0.1
q <- 1 - p
A <- c(5, 10, 20, 50, 100, 500)

# Generar grficos
par(mfrow=c(3, 2))  # Para organizar las 6 grficas

for (n in A) {
  x <- 0:n
  binomial_vals <- dbinom(x, n, p)  # Funcin de masa binomial
  normal_vals <- dnorm(x, mean=n*p, sd=sqrt(n*p*q))  # Aproximacin normal
  
  plot(x, binomial_vals, type="h", lwd=2, col="blue", ylim=c(0, max(binomial_vals, normal_vals)), 
       main=paste("n =", n), ylab="Probability/Density", xlab="x")
  lines(x, normal_vals, col="red", lwd=2)  # Aadir la normal
  legend("topright", legend=c("Binomial", "Normal"), col=c("blue", "red"), lty=1, lwd=2, xpd=TRUE, inset=c(0, +1.5))
}

Podemos notar como a medida que n crece, la Binomial se aproxima cada vez mejor a la Normal. Esto ya est establecido de entrada por el TML.

A medida que n crece, la funcin de masa binomial se aproximar cada vez ms a la curva normal, lo que visualiza el Teorema de Moivre-Laplace.

# Parmetros
p <- 0.1
q <- 1 - p
A <- c(5, 10, 20, 50, 100, 500)

# Generar grficos
par(mfrow=c(3, 2))  # Para organizar las 6 grficas

for (n in A) {
  x <- 0:n
  binomial_cdf <- pbinom(x, n, p)  # Funcin de distribucin acumulada binomial
  normal_cdf <- pnorm(x, mean=n*p, sd=sqrt(n*p*q))  # Aproximacin normal
  
  plot(x, binomial_cdf, type="s", lwd=2, col="blue", ylim=c(0, 1), 
       main=paste("CDF =", n), ylab="CDF", xlab="x")
  lines(x, normal_cdf, col="red", lwd=2)  # Aadir la CDF normal
  legend("bottomright", legend=c("Binomial", "Normal"), col=c("blue", "red"), lty=1, lwd=2, xpd=TRUE, inset=c(0, +1.5))
}

A medida que n crece, la CDF binomial se aproxima cada vez ms a la CDF normal. Esto ilustra grficamente el Teorema de Moivre-Laplace, que dice que la binomial converge a la normal conforme n aumenta.
